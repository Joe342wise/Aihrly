# Aihrly ATS API
ASP.NET Core Web API for Aihrly Applicant Tracking System.

## Overview
Team-side pipeline for recruiters and hiring managers to manage job applications through a hiring pipeline.

## Features
- Job postings management
- Application tracking
- Stage transitions with history
- Notes and scoring system

(In progress - will be completed as project develops)

## Entities
- TeamMember - recruiters and hiring managers
- Job - job postings
- Application - candidate applications
- ApplicationNote - notes on applications
- StageHistory - stage transition records
